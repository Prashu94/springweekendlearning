--spring.profiles.active="prod" --build.version=1.1 -> Program arguments
-Dspring.profiles.active="prod" -Dbuild.version=1.3 -> Vm Options
SPRING_PROFILES_ACTIVE=prod;BUILD_VERSION=1.4 -> EnvironmentVariables

Highest preference to VM Options
Between Program Arguments and EnvironmentVariables it will Program Arguments
If only EnvironmentVariables and without version then application.yml build version picked.
